# 介绍

### 简介

`sard-taro` 是一个使用 `React` 框架开发的适用于 `Taro` 的 UI 组件库。

目前 `sard-taro` 兼容 H5 / 微信小程序，后续会新增对其他小程序和 `react-native` 的兼容。

### 特性

- 🧩 50+ 高质量组件，覆盖移动端主流场景
- 💪 支持一套代码同时开发多端小程序+H5
- 🌿 支持按需引用
- 📖 详尽的文档和示例
- ʦ 支持 TypeScript
- 🌈 支持定制主题
- 🌍 国际化支持
- 🌙 支持暗黑模式
- 🧪 单元测试覆盖率超过 80%，保障稳定性
