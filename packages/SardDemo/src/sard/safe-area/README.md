# SafeArea 安全区域

### 介绍

添加一个空元素填充非安全区域。

### 引入

```ts
import { SafeArea } from 'sard'
```

## API

### SafeAreaProps

| 属性      | 描述           | 类型                                   | 默认值   |
| --------- | -------------- | -------------------------------------- | -------- |
| direction | 元素放置的位置 | 'top' \| 'right' \| 'bottom' \| 'left' | 'bottom' |
